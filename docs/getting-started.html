Getting Started

StellaraeSecure is free and open-source under the MIT License. Start by running the integrated stack locally and validating service health endpoints.

Prerequisites

  • Docker with Compose plugin
  • Git and a local clone of this repository
  • Node.js and Rust only if you want local non-container builds

1. Configure environment

Copy the compose template and set keys/secrets for your environment.

cp .env.compose.example .env.compose

2. Start stack

docker compose --env-file .env.compose up -d --build

3. Verify health

curl http://127.0.0.1:3000/healthz
      curl http://127.0.0.1:4100/healthz
curl http://127.0.0.1:4000/healthz
curl http://127.0.0.1:8080/healthz

Create your first account

Use the hosted signup flow: secure.stellarae.dev/signup
